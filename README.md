# AI SDR Agent

An AI-powered Sales Development Representative automation built with n8n, OpenAI, and Google Workspace.

## Current Features

- Fetches a company website
- Analyzes homepage content with OpenAI
- Extracts structured company research
- Stores results in Google Sheets

## Tech Stack

- n8n
- OpenAI
- Google Sheets
- GitHub

## Workflow

Manual Trigger → Edit Fields → Fetch Company Website → Analyze Company Homepage → Google Sheets
